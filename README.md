HardLinksRecursuve
==============

Вместо копирования скрипт создает жесткие ссылки рекурсивно.

Использование
=============

hlcopy.cmd "папка источник" "папка назначения"

Известные пробелмы
=============

Если внутри папки источника есть симлинк на файл то создается жесткая ссылка симлинка (странно)
